# My Flask App

Это простое веб-приложение на Flask, демонстрирующее использование шаблонов Jinja2. Приложение состоит из двух страниц: главной страницы и страницы "О нас", которые используют общий базовый шаблон.

## Структура проекта

```
/project_folder
    /templates
        base.html
        home.html
        about.html
    app.py
```

- **`app.py`**: Основной файл приложения Flask, содержащий маршруты для страниц.
- **`templates/base.html`**: Базовый HTML-шаблон, определяющий общую структуру страниц.
- **`templates/home.html`**: Шаблон для главной страницы, расширяющий `base.html`.
- **`templates/about.html`**: Шаблон для страницы "О нас", расширяющий `base.html`.

## Установка и запуск

1. **Клонируйте репозиторий:**

    ```bash
    git clone <URL вашего репозитория>
    cd project_folder
    ```

2. **Установите зависимости:**

    Убедитесь, что у вас установлен Python и pip. Установите Flask с помощью команды:

    ```bash
    pip install flask
    ```

3. **Запустите приложение:**

    Выполните следующую команду, чтобы запустить сервер разработки:

    ```bash
    python app.py
    ```

4. **Откройте приложение в браузере:**

    Перейдите по адресу `http://127.0.0.1:5000/`, чтобы увидеть главную страницу. Для перехода на страницу "О нас" используйте `http://127.0.0.1:5000/about`.

## Использование

- **Главная страница**: Содержит приветственное сообщение и навигацию.
- **Страница "О нас"**: Содержит информацию о приложении.

## Как это работает

Проект демонстрирует использование наследования шаблонов в Flask с помощью Jinja2. Базовый шаблон `base.html` содержит общие элементы страницы, такие как заголовок, меню навигации и футер. Страницы `home.html` и `about.html` расширяют `base.html`, добавляя специфический контент для каждой страницы.
